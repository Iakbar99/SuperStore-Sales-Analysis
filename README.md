#  Superstore Sales Analysis Project

## Project Overview

This project analyzes retail sales data to uncover insights on sales performance, profitability, and customer behavior.

---

##  Tools Used

* SQL (Data Analysis)
* Power BI (Dashboard & Visualization)
* Excel (Data Cleaning)

---

##  Key Business Questions

* Which products generate the highest profit?
* Which categories are loss-making?
* Which region performs best in terms of sales and profit?
* What are the monthly sales trends?

---

## Key Insights

* Furniture category shows lower profitability despite high sales
* Technology category contributes the highest profit
* Certain products are consistently loss-making
* Sales peak during specific months indicating seasonal trends

---

## Dashboard Features

* KPI Cards (Sales, Profit, Profit Margin)
* Region-wise performance analysis
* Category & sub-category breakdown
* Top products and loss-making products
* Interactive filters (Region, Category, Date)
